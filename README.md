# BAB Application - Backend

Ce projet présente la correction de bugs et l’amélioration prévu pour l'application de gestion de bibliothèque BAB : Book A Book.

# Implementation et correcifs de bugs (Release 1.0.0)

- Fixe sur l'expiration de reservation, issue #2
- Fixe sur le liste de livres par utilisateur qui n'a pas inclut les livres reservees plus que 21 avant, issue #3
- Fixe le endpoint /book/notget qui n'a pas marché avant, issue #7

# Plan des nouveautés et amelioration prévu pour nouvel version

- Ajouter le possibilité de inscrire et creer un compte via l'application
- Ajouter un endpoint pour afficher tous les livres considerée comme "non-disponible" tel ques "BORROWED" et "BOOKED"
